一个基于现代Spring技术栈的快速开发脚手架工具，帮助Java开发者快速构建安全、高效的企业级应用。

## 技术栈亮点

- **前沿技术**：基于Java 21 + Spring Boot 3.5.4 + Spring Security 6.5.2
- **安全认证**：基于Spring Security实现的完整的RBAC权限控制系统，开箱即用
- **现代化架构**：清晰的模块划分和代码组织结构

## 功能特性

### 已实现功能

- 认证鉴权
  - ✅ 基于RBAC的完整权限控制系统

  - ✅ 统一异常处理机制
